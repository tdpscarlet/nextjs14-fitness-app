## Fitme Fitness Web App
[Website](https://nextjs14-fitness-app.vercel.app)

Tech Stack:

- Typescript
- NextJS
- ReactJS
- TailwindCSS
- Redux Toolkit
- Prisma
- MySQL

Current features:
- Search exercise, food
- BMI, macros, ideal weight, body fat calculator
- Login, sign up
- Food diary
- Nutrition goal

Incoming features:
- Exercise diary
- Data visualize chart
- Email verification
- Forgot password
